# 🩺 SehatSaathi-AI

AI-powered healthcare triage web application that analyzes user symptoms using intelligent clinical reasoning and provides risk-level prediction along with nearby doctor recommendations for faster medical assistance.

---

## 🚀 Features

• AI-based symptom analysis  
• Risk level classification (Low / Medium / High)  
• Smart medical guidance in simple language  
• Automatic nearby doctor suggestion for high-risk cases  
• Optional nearby doctor suggestions for low & medium risk  
• Medical report parameter explanation (Hemoglobin, RBC, etc.)  
• Card-based clean UI for health insights  
• Emergency-support ready architecture  
• Multi-language response-ready system  

---

## 🧠 How It Works

• User enters symptoms in text form  
• AI analyzes symptoms using medical reasoning logic  
• System predicts severity level  
• Personalized guidance is generated  
• Nearby doctors suggested based on urgency

## ⚙️ Installation & Setup

### Step 1: Clone repository

git clone https://github.com/ayushbadola74/sehatSaathi-AI.git

### Step 2: Move into project folder

cd sehatSaathi-AI

### Step 3: Install dependencies

npm install

### Step 4: Run development server

npm run dev

---

## 🔐 Environment Variables Setup

Create a `.env.local` file inside project root:

VITE_OPENAI_API_KEY=your_api_key_here

⚠️ Never upload `.env.local` to GitHub
